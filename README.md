# MD Editor 使用说明

MD Editor 是一款桌面端 **Markdown 编辑器**，支持实时排版（所见即所得）、分栏预览和源码编辑，可管理文件夹、导出文档。本文说明 **安装已发布的软件** 以及 **日常操作**，面向使用者而非开发人员。

---

## 安装与启动

请从发布页获取对应系统的安装包（文件名与扩展名因平台而异）。

### Windows

- **安装版**：运行安装程序（如 NSIS 向导），按提示选择安装目录；安装完成后可从「开始」菜单或桌面快捷方式启动。
- **便携版**：解压或运行便携程序到任意文件夹，双击主程序即可使用，通常无需安装（不写注册表或写入较少，视具体发布包而定）。

若系统提示「未知发布者」，属未代码签名的常见提示，需自行判断是否信任来源后再运行。

### macOS

- 打开 **DMG**，将 **MD Editor** 拖入 **应用程序** 文件夹。
- 若首次打开被拦截，可在「系统设置 → 隐私与安全性」中允许，或右键应用选择「打开」。

### Linux

- **AppImage**：赋予可执行权限后双击运行，或使用终端执行该文件。
- **deb**：使用系统软件安装工具或 `dpkg` / `apt` 安装后，在应用菜单中启动。

## 界面与文件

### 菜单栏

窗口顶部为菜单：**文件**、**编辑**、**视图**、**主题**、**帮助**。无边框窗口时，菜单在自定义标题栏左侧。

### 文件与工作区

- **新建 / 打开**：在 **文件** 菜单中新建空白文稿，或打开单个 `.md` 文件。
- **打开文件夹**：打开一个目录作为工作区，便于在左侧 **资源管理器** 中浏览、切换同目录下的 Markdown。
- **保存**：**Ctrl+S** 保存；若尚未指定路径，会提示选择保存位置。**Ctrl+Shift+S** 可另存为。
- **多标签**：可同时打开多个文件；标签栏切换文稿。**Ctrl+Tab** 在标签之间循环切换（Windows 下由本软件使用该快捷键）。

### 关闭与未保存提示

关闭窗口或退出时，若有未保存修改，会提示是否保存。关闭单个标签前若该标签有未保存内容，也会提示。

## 编辑方式

可在 **视图** 菜单中切换，或使用快捷键 **Ctrl+E** 在三种模式间循环：

| 模式 | 说明 |
|------|------|
| **所见即所得** | 直接像文档一样编辑，格式即时呈现。 |
| **分栏预览** | 一侧 Markdown 源码，一侧预览效果。 |
| **源码模式** | 专注编辑纯文本源码，适合习惯键盘与语法的用户。 |

**查找与替换**：**Ctrl+F** 查找，**Ctrl+H** 替换（在所见即所得模式下使用查找/替换时，会自动切到适合操作的模式）。

## 常用快捷键

以下为 **Windows / Linux** 常用键；在 **macOS** 上多数可将 **Ctrl** 换为 **⌘（Command）**。

<!-- shortcuts:start -->

| 快捷键 | 作用 |
|--------|------|
| Ctrl+N | 新建文件 |
| Ctrl+O | 打开文件 |
| Ctrl+S | 保存 |
| Ctrl+Shift+S | 另存为 |
| Ctrl+W | 关闭当前标签 |
| Ctrl+E | 切换编辑模式（所见即所得 / 分栏 / 源码） |
| Ctrl+F / Ctrl+H | 查找 / 替换 |
| Ctrl+Shift+\ | 显示或隐藏 **大纲** 面板 |
| Ctrl+Shift+B | 显示或隐藏 **侧边栏** |
| Ctrl+Shift+F | **专注模式**（隐藏干扰元素，专心写作） |
| Ctrl+T | 打开 **表格** 插入工具 |
| Ctrl+Tab | 在已打开的标签之间切换（配合 Shift 可反向切换） |
| Ctrl+Shift+P | 循环切换内置 **主题** |

界面 **字号**（放大 / 缩小 / 重置）请在 **视图** 菜单中操作，未绑定默认全局快捷键。

更完整的说明可在软件内 **帮助 → 快捷键说明** 中查看。

## 主题与外观

在 **主题** 菜单中可切换 **亮色、暗色、护眼、Nord** 等内置配色。也可 **加载自定义 CSS** 文件以微调界面与预览样式；不需要时在菜单中清除自定义 CSS。

---

## 图片与导出

### 粘贴图片

将截图或图片 **粘贴** 进文档时，需要 **当前文稿已保存到磁盘**（有固定文件路径）。图片会保存到该文稿所在目录下的 **`images`** 文件夹中，并自动插入 Markdown 图片语法。若尚未保存，请先保存文件再粘贴图片。

### 导出文档

在 **文件** 菜单中可将当前文稿导出为 **PDF**、**HTML** 或 **Word（.docx）**，按提示选择保存位置即可。

---

## 大纲与状态栏

- **大纲**：根据标题生成目录，点击可跳转到对应位置；可用 **Ctrl+Shift+\\**（反斜杠物理键）开关。
- **状态栏**：显示编码、字数等信息，并提供部分视图选项入口。

---

## 卸载

- **Windows 安装版**：在「设置 → 应用」中卸载，或使用安装包提供的卸载入口。
- **macOS**：将 **应用程序** 中的 MD Editor 移入废纸篓。
- **Linux**：按对应包管理器方式卸载 deb，或直接删除 AppImage 文件。
